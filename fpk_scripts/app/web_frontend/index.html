❓ 使用帮助

快速开始

  • 插入 USB 设备后，设备卡片会自动显示
  • 点击设备卡片的「开始复制」按钮开始复制视频文件
  • 复制完成后，文件将自动进入 FTP 上传队列

WebSocket 连接

  • 页面通过 WebSocket 与后端实时通信
  • 连接断开时会自动重连（每 3 秒一次）
  • 顶部状态指示灯显示当前连接状态

快捷键

  • Esc - 关闭弹窗
  • S - 打开设置
